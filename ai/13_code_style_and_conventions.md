# Code Style

## Naming

- Classes: PascalCase
- Variables: camelCase
- Enums: PascalCase
- Files: snake_case.dart

## Formatting

- 100 character line limit
- No long methods (>50 lines)
- No nested logic > 3 levels

## Clean Code Rules

- No magic strings
- No duplicated logic
- Use extension methods when appropriate
- Avoid premature optimization
