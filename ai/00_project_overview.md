# Project Overview

## App Purpose
Mobile application that analyzes PDF bank statements from Guatemalan banks (Ficohsa, BAC, BI, Nexa), categorizes transactions, and generates financial analytics and PDF reports.

## Core Principles
- Offline-first
- No user accounts
- No cloud storage
- Local PDF processing
- Privacy by design
- One-time unlock monetization

## Supported Platforms
- iOS
- Android

## Main Flow
Upload PDF → Analyze → Categorize → Store → Show Dashboard → Export Report (Pro)

## Non-Goals (MVP)
- No backend
- No authentication
- No cloud sync
- No ML categorization
